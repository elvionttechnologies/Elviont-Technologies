# Elviont Technologies

Official GitHub Pages website for [elviont.nl](https://elviont.nl).

## Structure

- `index.html` - Main one-page company website.
- `betintel/` - BetIntel privacy and support pages.
- `unipilot/` - UniPilot privacy and support pages.
- `alealens/` - AleaLens privacy and support pages.
- `favicon.svg`, `favicon.png`, `favicon.ico`, `apple-touch-icon.png` - Browser and device icons.
- `CNAME` - Custom domain configuration for GitHub Pages.
- `.nojekyll` - Keeps GitHub Pages from processing the site with Jekyll.

## Live Routes

- Main site: `https://elviont.nl/`
- BetIntel privacy: `https://elviont.nl/betintel/privacy/`
- BetIntel support: `https://elviont.nl/betintel/support/`
- UniPilot privacy: `https://elviont.nl/unipilot/privacy/`
- UniPilot support: `https://elviont.nl/unipilot/support/`
- AleaLens privacy: `https://elviont.nl/alealens/privacy/`
- AleaLens support: `https://elviont.nl/alealens/support/`

## Deployment

The site is served by GitHub Pages from the `main` branch root.
